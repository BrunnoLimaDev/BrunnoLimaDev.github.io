Mockup da interface do Relatório Semanal de Valores

Visão Geral

O Relatório Semanal de Valores é uma aplicação web desenvolvida para gerenciar e rastrear contribuições financeiras semanais de equipes e sub-equipes. O sistema permite que usuários comuns registrem valores monetários e que usuários financeiros gerenciem relatórios e exportem dados em formato PDF. O projeto foi construído com foco em usabilidade, segurança e eficiência, utilizando tecnologias web modernas.

Tecnologias Utilizadas

  • HTML5 e CSS3: Estrutura e estilização da interface, com design responsivo e amigável.
  • JavaScript: Lógica do sistema, incluindo manipulação do DOM, validação de dados e gerenciamento de estado.
  • Cleave.js: Formatação de entradas monetárias para facilitar a inserção de valores no formato brasileiro (R$).
  • jsPDF: Geração de relatórios em PDF para exportação de dados semanais.
  • LocalStorage: Armazenamento local para persistência de dados entre sessões.

Funcionalidades Principais

  • Autenticação de Usuários: Sistema de login com diferentes níveis de acesso (usuários comuns e financeiros).
  • Registro de Valores: Usuários comuns podem adicionar valores monetários associados a sub-equipes, com validação e confirmação.
  • Gerenciamento de Sub-Equipes: Usuários financeiros podem criar sub-equipes para melhor organização.
  • Relatórios Semanais: Visualização de valores por semana, equipe e sub-equipe, com totais calculados automaticamente.
  • Exportação em PDF: Geração de relatórios detalhados em formato PDF para download.
  • Interface Intuitiva: Design limpo e funcional, com tabelas dinâmicas e mensagens de feedback ao usuário.

Desafios e Soluções

  • Desafio: Garantir a formatação correta de valores monetários no input.
    Solução: Integração do Cleave.js com tratamento robusto via expressões regulares para conversão de valores.
  • Desafio: Persistência de dados sem backend.
    Solução: Uso do LocalStorage para armazenar dados de forma eficiente e acessível.
  • Desafio: Validação de sub-equipes dinâmicas.
    Solução: Implementação de lógica condicional para lidar com equipes padrão e sub-equipes criadas dinamicamente.

Resultados

O sistema foi implementado com sucesso, oferecendo uma solução prática para o gerenciamento de contribuições financeiras. A interface amigável e as funcionalidades de exportação atenderam às necessidades do cliente, enquanto a correção de bugs (como problemas na adição de valores) garantiu maior confiabilidade. O projeto demonstra minha capacidade de desenvolver aplicações web completas, desde o front-end até a lógica de negócios, com foco em usabilidade e manutenção.
